CLASSIFICATION OF DATA STRUCTURE





   Built-In Data Structure:

      Built-In DS are simple data structures that can be constructed using primitive data types provided by computer languages.

   Types of built-in DS are:
        -Integer
        -Character
        -Float

   User-Defined Data Structure:

      User defined DS is a compound DS that can be constructed with the help of any one of the primitive data structure and it is having a specific functionality.
      As the name suggests, it can be designed by the user.

    Types of User-Defined Data Structure are:
        1.Linear
        2.Non-Linear
